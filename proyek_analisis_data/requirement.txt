detail
1.pandas (Python Data Analysis Library)
pandas adalah library Python yang digunakan untuk manipulasi dan analisis data. Library ini menyediakan struktur data seperti DataFrame dan Series, yang memudahkan dalam mengolah dan menganalisis data tabular (mirip dengan spreadsheet atau tabel SQL).

Fungsi Utama:
- Membaca/Menulis Data: Mendukung format seperti CSV, Excel, JSON, SQL, dan lainnya.
  Contoh: pd.read_csv('data.csv') untuk membaca file CSV.
- Manipulasi Data: Anda bisa melakukan operasi seperti filtering, grouping, merging, dan reshaping data.
  Contoh: df.groupby('column_name').sum() untuk mengelompokkan data berdasarkan kolom tertentu.
- Pembersihan Data: Mengatasi data yang hilang, duplikat, dan inkonsistensi data.
  Contoh: df.dropna() untuk menghapus nilai kosong.

2. streamlit
streamlit adalah framework Python yang memudahkan pembuatan aplikasi web interaktif, terutama yang berfokus pada visualisasi dan analisis data. streamlit sangat populer untuk membuat dasbor, aplikasi data science, dan machine learning dengan kode minimal.

Fungsi Utama:
- UI Components: streamlit menyediakan berbagai komponen antarmuka pengguna seperti tombol, slider, input teks, dan lainnya.
  Contoh: st.button('Klik Saya') untuk membuat tombol.
- Visualisasi: Mendukung berbagai grafik dengan pustaka lain (seperti matplotlib, plotly) yang mudah ditampilkan.
  Contoh: st.line_chart(data) untuk menampilkan grafik garis.
- Sederhana: Aplikasi dapat dijalankan hanya dengan satu perintah streamlit run app.py, dan tampilan langsung diperbarui.

3. matplotlib
matplotlib adalah library untuk membuat visualisasi data dalam bentuk grafik 2D. Library ini sangat fleksibel dan dapat digunakan untuk membuat berbagai macam plot, seperti bar chart, line chart, scatter plot, dan pie chart.

Fungsi Utama:
- Pembuatan Grafik: Anda dapat membuat berbagai jenis grafik dengan kontrol penuh terhadap elemen visual seperti warna, label, dan ukuran.
  Contoh: plt.plot(x, y) untuk membuat grafik garis.
- Kustomisasi Grafik: Setiap elemen grafik bisa dikustomisasi, seperti judul, sumbu, anotasi, dan lain-lain.
  Contoh: plt.xlabel('X Axis') untuk menambahkan label pada sumbu X.
- Integrasi: matplotlib bisa diintegrasikan dengan library lain seperti pandas untuk memvisualisasikan data dengan cepat.
